## 项目概述

本项目包含 **第四阶段（实时热门影视监控大屏）** 和 **第五阶段（个性化电影推荐系统）** 两个独立子项目，均可在 IDEA 中直接运行，无需任何大数据集群环境。

## 快速开始

### 环境要求

- JDK 8+
- Maven 3.6+
- IntelliJ IDEA

### 第四阶段：实时热门影视监控大屏

```bash
cd stage4-standalone
mvn package -DskipTests
java -jar target/realtime-dashboard-1.0.0.jar
```

访问 http://localhost:8080/dashboard.html

**功能特性：**
- 模拟 Kafka 实时点击数据流（每2秒自动生成）
- 启动时自动加载 `kafka_click_logs.csv` 历史数据
- 10秒滑动窗口聚合计算（每5秒刷新）
- 手动注入点击事件（点击电影按钮即可实时添加）
- 前端 ECharts 大屏每3秒自动轮询刷新

### 第五阶段：个性化电影推荐系统

```bash
cd stage5-standalone
mvn package -DskipTests
java -jar target/movie-recommendation-1.0.0.jar
```

访问 http://localhost:8081/recommend.html

**功能特性：**
- 启动时自动加载 movies.csv + ratings.csv 到 H2 数据库
- 纯 Java 实现的 ALS 协同过滤算法（无需 Spark）
- 为每个用户生成 Top-3 个性化推荐
- 离线分析页面（分类统计、评分排行、用户活跃度）
- 支持 H2 数据库控制台（http://localhost:8081/h2-console）

## 核心算法

### ALS 协同过滤

ALS（交替最小二乘法）将用户-评分矩阵分解为两个低秩矩阵的乘积：

```
R ≈ U × V^T
```

- **U**：用户隐语义特征矩阵
- **V**：物品隐语义特征矩阵
- **k**：隐语义维度（默认10）

训练过程交替迭代优化：
1. 固定 V，求解 U → 最小二乘
2. 固定 U，求解 V → 最小二乘
3. 重复1-2共10次
4. 预测评分 = U_i · V_j

参数配置：`rank=10, iterations=10, regParam=0.1`

## 数据集

| 文件 | 说明 | 数据量 |
|------|------|--------|
| movies.csv | 电影信息（ID/名称/分类） | 8部电影 |
| ratings.csv | 用户评分（用户ID/电影ID/评分/时间戳） | 14条评分 |
| kafka_click_logs.csv | 点击日志（电影ID/时间戳） | 8条记录 |

电影分类：Sci-Fi（科幻）4部、Drama（剧情）2部、Romance（爱情）1部、Comedy（喜剧）1部
